What the hack does ACID mean in database?
ad
Jul 23, 2018
One minute read

I’ve always wondered about the term ACID mean in database since I took the database course back in my bachelor degree.

In computer science, ACID (Atomicity, Consistency, Isolation, Durability) .

  • A stands for atomicity, meaning a transaction is treated as a whole which should either be success or fail.
  • C stands for consistency, meaning data stored in database should be queried in the same status aka value in any time.
  • I stands for isolation, meaning in the context of masive concurrency, data should be isolated from each queries even 2 transactions are modifying the same piece of data.
  • D stands for durability, meaning data should not get lost for any reason except manual operation.
