什么是postgis

postgis是基于postgresql开发的专门用于查询地理数据的插件。能够用sql语句实现很多原先需要很多复杂计算逻辑的地理数据查询功能，比如点和点之间的距离，路的长度，区域的面积等。

术语

SRID: spatial reference identifier，也就是我们平常说的坐标系统。
WKT: well-known text，postgis的基本数据的二进制数据的字符串表示。
WKB: well-known binary，postgis中真正保存的二进制数据。

基本数据类型

Geometry

其他地理数据的泛化类型，包含SRID表示该类型的地理数据在什么样的地理坐标系下。

Point:

几种得到point的方法 1. ST_GeographyFromText('SRID=4326;POINT(-43.23456 72.4567772)' 2. 'POINT(0 0)'::geometry作为函数的参数快速生成点。 3. ST_StartPoint(the_geom)给定某条线形geometry，得到开始点。 4. ST_EndPoint(the_geom)得到终止点。 5. ST_POINT(2.5,2.5)快速得到一个点 示例: SELECT AddGeometryColumn(‘public’, ‘my_geometries’, ‘my_points’, -1, ‘POINT’, 2); call AddGeometryColumn function to add column called my_points with geometry type of POINT with 2 dimensions and srid is -1 INSERT INTO my_geometries (name,my_points) VALUES (‘Home’,ST_GeomFromText(‘POINT(0 0)’));

Linestring:

几种实例化Linestring的方法： 1. 'LINESTRING ( 0 0, 0 2 )'::geometry 2. ST_GeographyFromText('SRID=4326;LINESTRING(-43.23456 72.4567,-43.23456 72.4568)') 3. ST_MakeLine(point, point, ....)

示例: SELECT AddGeometryColumn (‘public’, ‘my_geometries’, ‘my_linestrings’,-1,‘LINESTRING’,2); INSERT INTO my_geometries (name,my_linestrings) VALUES (‘Linestring Open’,ST_GeomFromText(‘LINESTRING(0 0,1 1,1 -1)’));

Polygons:

几种实例化polygon的方法： 1. ST_MakePolygon(linestring)

示例: SELECT AddGeometryColumn(‘public’,‘my_geometries’, ‘my_polygons’,-1,‘POLYGON’,2); INSERT INTO my_geometries (name,my_polygons) VALUES (‘Triangle’, ST_GeomFromText(‘POLYGON((0 0, 1 1, 1 -1, 0 0))’));
