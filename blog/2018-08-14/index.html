  • eq:
    comparing 2 objects equality. This is used when you want see if two objects meet the condition of a==b. For example:
class Province:
    def __init__(self, pname='', pcode=''):
        self.province_name = pname
        self.province_code = pcode
    def __eq__(self, other):
        return self.province_code==other.province_code
p1 = Province(pcode='111')
p2 = Province(pname='asd',pcode='111')
p1==p2 # will return true even the pname atributes are not the same.
