I’ve been using my blog system for quite a long time and find that the architecture of it happend to be very interesting. I think it is quite a good example to share for you ;) (Yeah, you dude!)

Here are the basic components that make up the blog system. * archtype framework: hugo * comment system: isso (containerized) * user tracking system: matomo (containerized) * theme: cocoa * server engine: nginx * cloud platform: AWS

I will explain each part respectively as I go further and especially the reason why I use them.

Hugo

Initially I use hexo as the static page framework which also support markdown. But turns out that the mathematics equation is not supported for hexo(at least for me, let me know if it can ;). So eventually I chose hugo which is written in GO.
