Starting from today, I decided to start learning courses in coursera and get some note about the coursed that I took.
Starting from the first one is Computational Thinking for Problem Solving.

In our daily life, we are facing different problems and in order to solve them, computational thinking could be a suitable way to do that.

  1. decomposition: devide a huge problem to sub managable problems.
  2. pattern recognition: finding similarities in sub problems. This will form the template of step in algo.
  3. data representation and abstraction: quantify charactors of objects inside sub problems. Abstracting variables is the key.
  4. algorithms: the chain or sequence of solution for each sub problems that one could execute later on.
