This post is a collection of what is the difference between blah blah and blah blah which is mainly about the terminology that I come across as a software engineer. I wanted to create content when learning. So I decided to start this topic and please help me append the content commenting below if you don’t mind ;)

  1. what is the difference between new and make in golang? new is going to initialize memory and return a pointer to the object.
    make is only used for initialize map, slice and channel. Plus do not return a pointer.

  2. what is the difference between object and class in scala? In scala, there are class, instance and object. object is a singleton

  3. what is the difference between 0.0.0.0, 127.0.0.1？ 0.0.0.0: points to all the ipv4 address of a host (one host might have different ipv4 addresses for internal access or outer access. For example, 10.xxx.xxx.xxx for external ip and 192.168.xxx.xxx for internal ip). Using this address for some sort of services that you started is allowing all of them to access this host. Alternatively, you can set the value for your service to listen on the address that you allow others to access like 10.0.1.1 for instance. 127.0.0.1: this is one of the {127.} family which is also called a loopback ip. Assigning this ip address to your service is telling it to only allow local access. Summary: Different kind of ip address is related to the host which has nothing to do with the remote accessing ip.
