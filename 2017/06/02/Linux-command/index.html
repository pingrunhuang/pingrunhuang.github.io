Linux command

This post is a continuous post which I will keep updating for those Linux commands that I think I tend to forget them.

top and ps for monitoring ps

在windows下加装ubuntu

  1. 下载rufus烧录usb安装盘
  2. 在windows下，右键计算机，点击管理，进行分区
  3. 重启后按要求安装，其中比较麻烦的是分区的过程，这里 有比较详细的教程，主要是一定要分出”/“,”/home”还有一个swap区，然后还有选择一个loader device， 直接选”/dev/sda”就好。
  4. 安装完成后，重启进入ubuntu，在terminal输入”sudo update-grub” 或者下载startupmanager进行操作。

  • help test: check for more condition that are testable

check the port for a certain process

  1. lsof -i -n -P
  2. sudo netstat -tlpn: -t means limit the output to TCP connections, -l for listening ports, -p list the program name and -n shows the numeric version of the port instead of a named version.
