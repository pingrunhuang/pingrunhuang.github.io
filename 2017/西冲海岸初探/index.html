西冲海岸初探
ad
Oct 1, 2017
One minute read

国庆节到处都是人，去哪里好呢？这个国庆，决定把深圳最好玩的野场西涌搞个透彻，不然都不好意思以后都不好意思带人去深圳最好玩的地方了。

怎么去西冲

有车的土豪可以选择开车，但个人不太建议新手开车去，或者应该错开节假日高峰期。因为上山有一段路来回都是单行道，很多老司机会喜欢窜道， 这队医新手来说是很危险的事情，因为一旦发生意外，后面就会堵死了。所以为了你我他，我建议还是选择公交线路。说到公交线路，我终于找到了一条安全快捷的线路了！ 深圳北->E11->南澳街道办->选择拼车或者坐公交（听说有个专线2可以直达，个人没试过）-> 鹤薮村 ->抵达客栈聚集地。这里就是客栈或者民宿散落的地方了，各种各样的民宿任君挑选。

住

选择住宿就要看你来的目的了，像我是打算来冲浪的，当然是选择靠近沙滩的民宿啦。就我所知，西冲有三个村落是民宿聚集地而且靠近3，4号沙滩的， 分别是鹤薮村，新屋村和芽山村。那么问题就来了，哪些民宿靠近沙滩呢？很不幸的是，这次我因为一些原因选择了较远的村落–芽山村。这里的坏处也是他的好处就是，相对沙滩比较远，但人也相对少，因此晚上会比较安静。要知道，来这里度假的一般是家人或者团体。团体很喜欢租一个大别墅然后在里面喊歌，这真的很干扰人休息好吗。其次就是鹤薮村了，而经过我实地勘查，最近的还是新屋村距离沙滩大概才1公里，这是直线距离哦。

冲浪

好了，重点来了，冲浪！我是一个喜欢滑雪，滑板运动的人，我很喜欢踩着板子滑出去的感觉，很畅快，如果可以滑出美妙的曲线和动作，会有一种莫名的满足感。 那么，冲浪就肯定是我必学的技能啦！这是我在沙滩边上看到的几家冲浪店，所以其实没必要提前预定的，到了沙滩后直接上去问价，租板或教学都可以。 虽然我没有都进去看，但问了几家，价格大概都是团体400多，一对一500多的样子。我报了一个460的体验课，因为我觉得以我个人的运动天赋来说，不需要一对一，我只需要有个人示范而且可以问一下问题就好，所以选什么还是要看个人需求啦。个人认为报名不需要提前报，只需要去到沙滩，那里会有很多家俱乐部等你进去询价，easy！ 这是我报名的那家俱乐部，就在沙滩旁边很显眼的绿草坪上，感觉很多浪客会来这里聚集。 这是他家的傻狗，真心傻，只知道吃。。。

回家后搜到的将要去的浪点

  1. 台湾垦丁
  2. 澳大利亚黄金海岸(GoldCoast)
  3. 菲律宾Siargao
  4. 斯里兰卡
  5. 印尼库塔
  6. 巴厘岛canggu Among all these spots, I think I will go 垦丁 first，since it is closer to china and hence easy to reach out. Then Siargo cause it seems like siargo is suitable for beginner and the wave is soooo good!
