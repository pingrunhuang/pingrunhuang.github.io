DataFrame reset_index() method

For df with multiple-level index(level is column in the index group), return a new df with labeling info in the columns under the index names. When we use groupby() method with a argument with a list which means we are going to group by several columns. In this case, the result will have an index containing multiple columns. If we are going to store the df into database with the index as normal columns without unindexing the index, python will throw an e mysql.connector.errors.ProgrammingError. So we have to apply reset_index method to reset index. Below are some argument specification to remember:

  • level: which level(column) you want to reset,
  • drop : drop the index or not.
  • inplace: modify the df in place.
  • col_level: when I use it, it is creating a new column called ‘index’ and transform all the columns in the index to normal columns.
  • col_fill: not sure yet.
  • ‘df.append()’: append another df and return a new df
