Spark java api notation

I have been spending quite a lot of time learning how to process big data on a distributed system. Spark comes to the first solution. And I also wrote a sample application to process the movie data. I know we can simply use Scala for less code. Here is a demonstration on how to use Scala API. But for now, let’s just use Java Api. And I decided to write a blog to keep a track on what I learned from the Spark framework and how I used it.

The backbone of spark is RDD(Resilient distributed data set) which is an immutable data structure. Api caller can do Actions(reduce(), count(), first(), foreach()) and Transformations(map(), filter(), sample(), union()) on it.

Spark used a lot of Tuple. From Tuple1 to Tuple21. The number behind it is the number of element inside a tuple.

How to generate RDD?

There are 3 ways to create a RDD. All require creating a Spark Context first by using

  • Create from Collection object:
    JavaRDD<String> rdd1 = ctx.parallelize(list)
  • Create from Local file
  • Create from reading file in HDFS: JavaRDD<String> rdd = ctx.textFile("hdfs_path")

Grouped by Key

Say we want to group all the records that have the same key into one record. We can use
JavaPairRDD<String, Iterable<Integer>> grouped2 = rdd.groupByKey(); which result in key1 [value1, value2, value3...].

Map value

Say we want to apply a certain rules on the value but keep the key unchanged. For example, we want to only keep the max value in the list on the previous result. We can call rdd.mapValues() to achieve this.

Reduced by key

If we want to aggregate all the record with the same key. For example, count the record with the same userid. We can call rdd.reduceByKey(new PairFunction<T,K,V>(){...}) where T is the input record, K is the key and V is the value of the record.

Combine by key

The combineByKey function is an optimization of the groupByKey function by reducing the data transfer through network. Checkout explanation here.
The combineByKey function takes in 3 functions:

  • createCombiner function is going to initialize a combiner.
  • mergeValue merges a newly value into the previously collected values
  • mergeCombiner combines the merged values together into a single value.
