How to execute java program from Terminal?

Most of the time, we are running java program from IDE such as IDEA Intellij, eclipse .etc. But it is more convenient to run the java program from terminal (Here we only talk about Unix-like operating system). For example, when we just download an open source project which happen to have some demo program for demonstrating the usage of the library briefly, in this case we would like to check that little piece of code to see if it is really suitable for our need. More over, some program might require input that is consumed by the main method. So it would be better to run it from terminal for that specify class instead of rebuild a new project in IDE and import bunch of external libraries.

To make things more practical, I will use the case that I recently met when I tried to explore Lucene. You can download a distribution from the Lucene’s official website. And there will be a demo directory that has two classes for having a taste of Lucene. The -cp token specify the CLASSPATH of this program. You can set it as an environment variable. But that will mess up other programs so we better specify them as a prefix for each program. Here we specify the external jar file directories which are imported into this program. There is a . at the end to let java search for the current directory and its sub-dictionary.
Notice the directory of the library should be the full directory. For example, in mac, you can’t use ~/ to indicate the home directory. Instead, you should use /Users/username/ to indicate the home directory of the username user. I try many time using ~/ only getting Error: Could not find or load main class Main error.
